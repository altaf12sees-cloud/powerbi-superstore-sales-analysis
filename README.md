# powerbi-superstore-sales-analysis
Superstore Sales Performance Analysis (Power BI). This project analyzes a retail superstore dataset to uncover insights into sales performance, profitability, and customer behavior.
🎯 Objective

To identify key business trends, top-performing products, and areas of improvement to support data-driven decision-making.

## 📂 Dataset

* Source: Kaggle Superstore Dataset
* Includes: Orders, Sales, Profit, Customers, Regions

## 🛠 Tools Used

* Power BI
* DAX (Data Analysis Expressions)
* Data Cleaning (Power Query)

## 📊 Key Insights

* Sales show strong growth during Q4 due to seasonal demand
* A small group of customers contributes to a large portion of revenue
* Some products generate high sales but negative profit
* Discounts significantly impact profitability

## 📈 Dashboard Features

* KPI tracking (Sales, Profit, Growth %)
* Regional performance analysis
* Customer segmentation
* Profitability analysis
